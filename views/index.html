Easy contact forms, No setup.

Have you ever tried to make a contact form for your website? If you have, you'll know that it involves HTML for the form, Javascript for the verification, and PHP for the actual sending. It's a pretty big task, and if you use one of those "easy form" services, you sacrifice customizability. You're forced to use their drag-and-drop editor, which lacks any advanced features whatsoever. What if you could create a form in HTML, and someone else would take care of the rest? Sounds awesome? Welcome to FormMailer.

Bring Your Own Form

FormMailer is unlike other services in that we don't oversimplify things. There's no WYSIWYG editor, meaning that you can make your form however you like. CSS styles? No problem. Custom jQuery scripts and plugins? Great! We let you create your form, while we take care of the rest. It's that easy.

Server-Side Validation

FormMailer has built-in server-side form validation, and if we're doing it, that means you don't have to. When a POST request is made, the API will first validate the request and issue a response with the reason if it doens't pass vaildation. And all of this is optional, so you can disable it if you like.

Powered by Mandrill

FormMailer is powered by Mandrill. This means that you get tons of awesome features like templating, customization, and analytics. It's free for quite a few emails, and it's really easy to set up. Check out their website for more information.
